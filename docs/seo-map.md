- **Accueil** reçoit les liens « agence web Nantes » à forte autorité (menu, footer, contenu éditorial).
- **Pages métiers spécifiques** reçoivent des ancres descriptives depuis la page artisan générique.
- **Pages locales** renvoient vers services (création, SEO, location) avec ancres variées.
- Éviter les ancres génériques seules (« En savoir plus ») sur les pages money.
